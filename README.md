"anti-swear-browser-extension"
("text replacer")

Браузерное расширение для автоматической замены ненормативной лексики
на веб-страницах.

Возможности:
- Замена ненормативных слов
- Цензура текста
- Работа с динамическим контентом
- Настраиваемый словарь

Используемые технологии:
- JavaScript
- Chrome Extensions (Manifest V3)

Установка:
1. Скачайте репозиторий
2. Откройте chrome://extensions/
3. Включите режим разработчика
4. Нажмите «Загрузить распакованное»
5. Выберите папку проекта
